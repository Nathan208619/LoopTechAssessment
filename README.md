# Playwright Environment Setup

## Prerequisites

Install Node.js:

- https://nodejs.org/

Verify installation:

```bash
node -v
npm -v
```

---

# Install Playwright

From the root of your existing project, run:

```bash
npm init playwright@latest
```

Follow the setup prompts.

---

# Install Browsers

If browsers are not installed automatically:

```bash
npx playwright install
```

---

# Run Playwright Tests

```bash
npx playwright test
```
